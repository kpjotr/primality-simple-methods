-- primality_simple_methods (psm) repository. --<br>

Content of the repository: python implementations of simple methods of primality tests within my primefinders project.
There is no particular scientific goal, I just want to write a program that runs without intervention in the background, continuously archived and searches for prime numbers without an upper limit. I will try to find a solution as quickly as possible.

resources for my work:
https://en.wikipedia.org/wiki/Primality_test#Simple_methods


A repository taralma: primfinder projectemen belül prímtesztekre naív módszerek python implementációi.
Nincs különösebb tudományos cél, csak szeretnék egy olyan programot írni, mely beavatkozás nélkül fut a háttérben, folyamatosan archiválva keresi a prímszámokat felső korlát nélkül. Igyekszem a lehető leggyorsabb megoldást megtalálni.
